# Elomaze

Elomaze is a marketplace platform that connects users with agents and service providers to help them find accommodation, services, and student-related needs easily.

## Features

- Property and service listings
- Real-time chat between users and agents
- Review and rating system
- Community-based insights (future feature)

## Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Modules

- Authentication
- Listings (services/properties)
- Chat system
- Review system
